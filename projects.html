Description:

A point of sales system for a nail salon called Gorgeous Nails. It allows customer checkout for services they want (e.g. nails, pedicure, etc.) and keeps track of orders. There is an admin page included (with username and password authorization) that leads to the admin page with various functionality: add, remove, and edit employee and service details. Functionality also includes viewing all orders.

Used software technology:

    HTML

  • Initial index.html (home page) of project
  • Admin login and admin page but later converted to embedded javascript templating (EJS) file format

    • CSS

  • Styling of elements, including formatting, visual display, and hovering

    • JavaScript

  • Document Object Model (DOM) manipulation
  • Click functionality of elements (buttons, dropdown lists, page tabs)

    • JQuery

  • Used similarly to JavaScript, and used initially but ended up using JS more often
  • Specifically needed for Bootstrap modal manipulation (didn't work with just JavaScript)

    • Node.js

  • Acted as initial local server for development
  • Node package manager (npm) used to install packages (e.g. Express, Mongoose)

    • MongoDB

  • NoSQL database to hold information for various collections: orders, nails services, pedicure services, employees and managers

    • Mongoose

  • Backend library for MongoDB and Node.js to handle modeling of collections in MongoDB database for project

    • Express.js

  • Node.js framework to handle routes, as well as handling requests and views

    • Visual Studio Code

  • Utilized integrated development environment for coding
